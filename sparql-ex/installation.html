# Installation

The SPARQL.ex Hex package (opens new window) can be installed as usual, by adding sparql to your list of dependencies in mix.exs:

def deps do
  [
    {:sparql, "~> 0.3"}
  ]
end

The SPARQL.Client Hex package (opens new window) can be installed by adding sparql_client to mix.exs:

def deps do
  [
    {:sparql_client, "~> 0.4"}
  ]
end

See also the SPARQL.Client configuration page on how to setup the HTTP client to be used by SPARQL.Client.
