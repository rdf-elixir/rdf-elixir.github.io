# Limitations

  • the following ShEx features are not implemented yet:
    • invalid shape expression references in negations are not checked
    • imports
    • external shapes
    • annotations
    • semantic actions
  • greedy matching
  • non-ascii characters in regular expressions are not fully supported yet
